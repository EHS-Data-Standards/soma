    Outcomes Working Group Schema

    A LinkML data model for representing biological measurements, assays, and experimental protocols in the context of outcomes research. This is the main entry point that imports the measurement base schema and domain-specific microschemas.

    URI: https://w3id.org/EHS-Data-Standards/outcomes_working_group

    Name: outcomes_working_group
